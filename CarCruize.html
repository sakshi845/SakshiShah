Design of an Automobile Cruise Control system

This project designs a state-feedback controller for a car cruise control system using control engineering principles. The open-loop transfer function was derived and analyzed to determine the desired closed-loop behavior. A desired characteristic equation was computed based on desired parameters, and a state-feedback controller was designed to achieve the targeted car dynamics.

Software Used:
MATLAB, Simulink
Concepts Learned:
Communication Protocols, PLC Programming, Sensor and Actuator Integration, Digital Twin, Machine Learning-based Data Management, IoT, AI/ML in Predictive Maintenance, Cyber-Physical Systems
