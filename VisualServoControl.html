Visual Servo Control of a 7-Link Kinova Gen3 Robot Arm

My role involves developing a closed-loop control system to track a micro swimmer using an ultrasound probe and a 7-link Kinova Gen3 robot arm, as it navigates a viscous medium under vibration forces.

  • Developed a closed-loop control system to track dynamic objects in real-time using an eye-in-hand camera, leveraging Simulink, ROS, and OpenCV.
  • Designed and fabricated custom End-of-Arm Tooling (EOAT) in SolidWorks to stabilize the ultrasound probe and support a monocular camera.
  • Implemented real-time image processing and control algorithms (PID, MPC) using MATLAB and GStreamer, reducing system latency and enhancing tracking precision.
    • Software Used:
    MATLAB, OpenCV, GStreamer, ROS, Kinova Gen 3 Robot, Data-Driven Controllers
    Concepts Learned:
    Camera Calibration, Joint Space and Task Space Control, Trajectory Planning, Inverse Kinematics, Forward Kinematics, Object Detection, Jcobian, DH parameters, Interaction Matrix, API connections
