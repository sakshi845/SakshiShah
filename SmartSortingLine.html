Remote Control and Monitoring of Conveyor System for Package Delivery

This project focussed on developing a system taht Sorts packages by color using Micro800 PLC, with the following features: • Real-time data via Modbus TCP/IP and MQTT. • Data logging: PostgreSQL, Monitoring: Flask app on Render. • HMI for local control (status, start/stop, alerts). • ML model estimates processing time

Technologies Used: HTML, Connected Components Workbench, Python, VS Code, Factory I/O, Datagrip, PostgreSQL database, Render, Allen-Bradley Micro 800, GitHub

Concepts Learnt: Communication Protocols, PLC Programming, Sensor and Actuator Integration, Digital Twin, Machine Learning, based data management, IOT, AI/ML in predictive maintainance, Cyber Physical Syatems
